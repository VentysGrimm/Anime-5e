# Anime 5e Beyonder Worlds

Standalone source-backed Anime 5e module for the Beyonder Worlds genre expansion.

## Contents

- Character options: Beyonder Worlds Species, Classes, Attributes, Defects, and modern/future skills.
- Items: science fiction weapons and armour, cyberpunk gear, spy tech, modern weapons, vehicles, starships, and mecha.
- Actors: science fiction, mecha fantasy, urban fantasy, and cyberpunk threats.
- Journals: genre guides and configuration notes for keeping modern/future content out of fantasy-only base packs.

Validate from the base system root with:

```text
node tools/validate-content-module.mjs modules/anime5e-beyonder-worlds
```
